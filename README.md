# Zoppi Engenharia

Site institucional para venda de projetos de engenharia, planos de rigging, içamentos, andaimes, linhas de vida, laudos, treinamentos e produtos técnicos.

## Rodar localmente

```bash
npm start
```

Depois acesse:

```text
http://localhost:3000
```

## Deploy

O projeto pode ser publicado como site estático no GitHub Pages ou como aplicação Node no Railway usando:

```bash
npm start
```
